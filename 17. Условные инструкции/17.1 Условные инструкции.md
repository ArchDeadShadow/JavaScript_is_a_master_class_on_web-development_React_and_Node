## Инструкция `if`

```js
// Зарезервированное слово "if".
if (условие) {
  // Блок кода/иснтрукций, выполняемый однократно, если Условие правдиво
}
```

Пример:

```js
let val = 10;

// Это условие правдиво
if (val > 5) {
  val += 20;
}

console.log(val);
```

Пример `if` с оператором отрицания

```js
const person = {
  age: 30,
};

// !undefined === true
if (!person.name) {
  console.log("Имя не указано");
  // Другие действия в случае, если свойства "name" у обьекта "person" нету
}
```
