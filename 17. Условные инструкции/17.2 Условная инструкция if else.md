# Условная инструкция if else

```js
if (условие) {
  // Блок кода, выполняемый однократно, если Условие правдиво
} else {
  // Блок кода, выполняемый однократно, если Условие ложно
}
```

Пример:

```js
let val = 10;

// Это условие ложно
if (val < 5) {
  val += 20;
} else {
  // Выполнится этот блок инструкций
  val -= 20;
}

console.log(val);
```
