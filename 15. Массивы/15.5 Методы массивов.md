# Методы массивов

| Методы массивов: |         |         |
| ---------------- | ------- | ------- |
|                  | push    | pop     |
|                  | shift   | unshift |
|                  | forEach | map     |

Методы массивов - эта функции высшего порядка, либо методы прототипов.

Все эти методы мутируют массив, они не создают новый массив, они изменяют текущий массив.

## Функции высшего порядка в массивах:

`push()` - Добавление нового элемента в конец массива

```js
const myArray = [1, 2, 3];
console.log(myArray);

myArray.push(4);
console.log(myArray);

myArray.push(true);
console.log(myArray);
```

`pop()` - Удаление последнего элемента с конца массива, возвращает удаленный элемент

```js
const myArray = [1, 2, 3];
console.log(myArray);

myArray.pop();
console.log(myArray);

const removeElement = myArray.pop();
console.log(myArray);
console.log(removeElement);
```

`unshift()` - Добавление элемента в начале массива

```js
const myArray = [1, 2, 3];
console.log(myArray);

myArray.unshift(true);
console.log(myArray);

myArray.unshift("abc");
console.log(myArray);
```

`shift()` - Удаляет первый элемент массива, возвращает удаленный элемент

```js
const myArray = [1, 2, 3];
console.log(myArray);

myArray.shift();
console.log(myArray);

const removeElement = myArray.shift();
console.log(myArray);
console.log(removeElement);
```
