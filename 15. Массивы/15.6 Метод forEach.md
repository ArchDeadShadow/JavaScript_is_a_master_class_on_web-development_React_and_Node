# Метод forEach

`forEach()` - это перебор всех элементов массива и выполнение каких-то действий с этими элементами

```js
const myArray = [1, 2, 3];
console.log(myArray);

// Выполнение действий с каждым элементом массива
myArray.forEach((el) => console.log(el * 2));

// Метод forEach не меняет оригинальный массив
console.log(myArray);

/*
 * Метод forEach() - возвращает undefined,
 * единственная его задача это перебор элементов массива и выполнение действия с каждым элементом
 */
const res = myArray.forEach((el) => console.log(el * 2));
console.log(res);
```
