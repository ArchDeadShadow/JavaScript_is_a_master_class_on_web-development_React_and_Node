# Оператор сравнения для массивов и обьектов

Каждый масссив - это обьект, а каждый обьект - это ссылочный тип

```js
const myArray = [1, true, "Sergey"];
myArray;
myArray.length;

const myArray2 = [1, true, "Sergey"];
myArray2;
myArray.length;

myArray === myArray2; // false

const myArray3 = myArray;

myArray === myArray3; // true
```
