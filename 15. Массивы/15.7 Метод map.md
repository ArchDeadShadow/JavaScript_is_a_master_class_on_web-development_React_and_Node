# Метод map

`map()` - Ожидает callback функцию в качестве аргумента, также перебирает элементы, но **_возвращает_** новый массив.

```js
const myArray = [1, 2, 3];
console.log(myArray);

const newArray = myArray.map((el) => el * 3); // Возвращает новый массив

console.log(newArray);
console.log(myArray); // Оригинальный массив не изменился
```
