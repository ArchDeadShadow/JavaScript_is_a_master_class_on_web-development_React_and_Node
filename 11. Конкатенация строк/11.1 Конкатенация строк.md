# Конкатенация строк

Оператор + для конкатенации строк

```js
"Hello " + "World";
```

Переменные в конкатенации строк

```js
const hello = "Hello";
const world = "World";

// Добавление пробела между словами
const greeting = hello + " " + world;

console.log(greeting);
```

## Шаблонные строки

```js
const hello = "Hello";
const world = "World";

// Выражение между обратными кавычками
const greeting = `${hello} ${world}`; // Здесь может быть любое выражение, например, вызов функкции

console.log(greeting);
```

```js
const myName = "Сергей";
const myCity = "Нововолынск";

const whereILive = `Я ${myName} живу в городе ${myCity}`;

console.log(whereILive);
```
