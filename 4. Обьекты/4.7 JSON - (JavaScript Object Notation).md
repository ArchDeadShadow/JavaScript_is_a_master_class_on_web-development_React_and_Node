# JSON - JavaScript Object Notation

```javascript
{
	"userId": 1,
	"id": 1,
	"title": "Test Title",
	"status" {
		"completed": false
	}
}
```

Это формат обмена данными, передача данных в виде строки. Перевод в JS обьект осуществляется через метод `JSON.parse()`. Обратная конвертация обьекта JS в JSON через метод `JSON.stringify()`

```javascript
const post = {
  title: "My post",
  likesQty: 5,
};

JSON.stringify(post);

const postStringify = JSON.stringify(post);

JSON.parse(postStringify);
```
