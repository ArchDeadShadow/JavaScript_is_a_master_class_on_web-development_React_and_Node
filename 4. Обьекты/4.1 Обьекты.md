# Обьекты

Практически все сущности в js это обьекты.

Обьект - это набор свойств "имя: значение".

Обьект - это тип значений.

Обьект - это тип переменных.

Обьект - это ссылочный тип значений.

```javascript
const myCity = {
  city: "New York",
  popular: true,
  country: "USA",
};
```

Получение значений свойств. Точечная запись.

```javascript
console.log(myCity.city);

console.log(myCity.popular);

// Точечная запись изменения свойств
myCity.city = "Las Vegas";

console.log(myCity);

// Добавление новых свйоств через точечную запись

myCity.state = "NY";

console.log(myCity);
```
