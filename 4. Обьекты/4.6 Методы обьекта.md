# Методы обьекта

Метод - Это свойство обьекта, значение которого функция.

Свойства - не содержат функции.

```javascript
const myCity = {
  // Свойства
  city: "New York",
  // Метод
  cityGreeting: function () {
    console.log("Greetings!!!");
  },
};

// Вызов метода
myCity.cityGreeting();
```

Сокращеный формат записи методов

```javascript
const myCity = {
  // Свойство
  city: "New York",
  // Метод
  cityGreeting() {
    console.log("Greetings!!!");
  },
};

// Вызов метода
myCity.cityGreeting();
```
