Обьекты - Добавление новых свойств

```javascript
const myCity {
	city: "New York"
};

myCity.popular = true;

console.log(myCity);

myCity.country = 'USA'

console.log(myCity)
```

Удаление свойств через оператор delete.

```javascript
const myCity = {
  city: "New York",
  popular: true,
  country: "USA",
};

delete myCity.country;

console.log(myCity);
```

Добавление к значению свойств с использованием скобок (bracket notation)

```javascript
const myCity = {
  city: "New York",
};

myCity["popular"] = true;

console.log(myCity);

const countryPropertyName = "country";

myCity[countryPropertyName] = "USA";

console.log(myCity);
```
