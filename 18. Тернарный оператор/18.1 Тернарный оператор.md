# Тернарный оператор

У тернарного оператора три операнда. Конструкция с тернарным оператором - выражение. Выражение возвращает значение.

```js
/* Выражение. В условии может быть любое выражение. Если условие правдиво, тогда возвращается результат Выражения 1.
* Если условие ложно, тогда возвращается результат Выражения 2
*/
Условие ? Выражение 1 : Выражение 2
```

Рекомендованная запись тернарного оператора:

```js
// Более предпочтительный формат записи
Условие
	? Выражение 1
	: Выражение 2
```

Пример 1:

```js
const value = 11;

// Выражение-инструкция
value ? console.log("Условие истинно") : console.log("Условие ложно");
```

Пример 2:

```js
const value1 = 11;
const value2 = 25;

value1 && value2 ? myFunction1(value1, value2) : myFunction();
```

Пример 3:

```js
let value = 11;

// Возвращает значение
console.log(value >= 0 ? value : -value);

value = -5;
// Присваивание результата переменной
const res = value >= 0 ? value : -value;
console.log(res);
```
