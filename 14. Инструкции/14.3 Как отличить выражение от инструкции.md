# Как отличить выражение от инструкции

_Выражения_ - могут быть использованы как аргументы в вызовах функций

Инструкция или выражение?

Пример:

```js
function myFn(a) {
	console.log(a);
}

const b = true;
let c = 10;

myFn(5); // 5
myFn(b); // true
myFn(c = c + 1); // 11
myFn(c = c + 1); // Uncaught SyntaxError
myFn(let d); // Uncaught SyntaxError
```
