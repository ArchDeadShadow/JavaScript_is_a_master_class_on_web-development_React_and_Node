# Инструкции

В JS есть: выражение, инструкция, выражение - инструкция

Выражение - всегда возвращает значение

Инструкция - всегда выполняет действия

Примеры инструкции:

```js
let a;

const b = 5;

if ((a, b)) {
  console.log("a is larger");
} // Точка с запятой не требуется

for (let i = 0; i++; i < 5) {
  console.log(i);
} // Точка с запятой не требуется
```

_Инструкция_ обычно заканчивается точкой с запятой.

Исключение: точка с запятой не требуется после блока инструкций

Точку с запятой можно опускать
