# Преимущества и недостатки одностраничных приложений

**_Преимущества:_**

- Веб страница _не перезагружается_ при навигации по приложению
- _Моментальная_ перерисовка элементов веб страницы
- Легче разворачивать, потому что SPA содержит только _несколько статических файлов_
- Эффективное использование пропускной способности

**_Недостатки:_**

- Для поисковой оптимизации **_(SEO)_** необходимо выполнить дополнительную настройку сервера для корректного сканирования ботами, такими как Googlebot
- **_Большой размер файла_** JS влияет на время начальной загрузки приложения
