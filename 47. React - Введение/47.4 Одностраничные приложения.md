# Одностраничные приложения

Одностраничные приложения **_(SPA - Single Page Application)_**

Одностраничные приложения клиент запрашивает _только одну_ HTML страницу и JS, CSS файлы для всего приложения.

Фреймворки и библиотеки для создания _одностраничных_ приложений:

- React
- Angular
- Vue
- Svelte
- Ember
- Ruby
