# Обработка ошибок

Что происходит в случае ошибок:

```js
const fnWithError = () => {
  throw new Error("Some error");
};

// Выполнение кода остановиться после обьявления Uncaught ошибки
fnWithError();

console.log("Continue...");
```

## TRY/CATCH

```js
try {
  // Выполнение блока кода
} catch (error) {
  // Этот блок выполняется в случае возникновения ошибок в блоке try
}
```

Пример:

```js
const fnWithError = () => {
  throw new Error("Some error");
};

try {
  fnWithError();
} catch (error) {
  console.error(error);
  console.log(error.message);
}

console.log("Continue..."); // Выполнение кода продолжится
```
