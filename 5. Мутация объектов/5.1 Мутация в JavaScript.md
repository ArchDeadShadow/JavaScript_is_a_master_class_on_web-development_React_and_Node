Мутация в JavaScript

Значения примитивных типов

```javascript
const a = 10;

// Копируем значения (copy by value)
let b = a;

b = 30;

console.log(a);

console.log(b);
```

Значение ссылочного типа

```javascript
const person = {
  name: "Bob",
  age: 21,
};

// Мутация обьекта, на который ссылается переменная "person"
person.age = 22;
person.isAdult = true;

console.log(person.age);
console.log(person.isAdult);
```

Мутация копий обьекта

```javascript
const person = {
  name: "Bob",
  age: 21,
};

// Копирование ссылки (copy by reference)
const person2 = person;

// Мутация обьекта, на который ссылается "person2"
person2.age = 26;
person2.isAdult = true;

// Мутация также отразилась на переменной "person"
console.log(person.age);
console.log(person.isAdult);
```
