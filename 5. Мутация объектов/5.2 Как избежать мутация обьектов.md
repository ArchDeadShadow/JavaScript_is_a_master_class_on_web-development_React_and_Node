Как избежать мутация обьектов

Вариант 1

```javascript
const person = {
  name: "Bob",
  age: 25,
};

// Если у обьекта есть вложенные обьекты, то ссылки на них сохраняются
const person2 = Object.assign({}, person);

person2.age = 26;

console.log(person2.age);
console.log(person.age);
```

Вариант 2

```javascript
const person = {
  name: "Bob",
  age: 25,
};

// ... - Оператор разделения обьекта на свойства
// Ссылки на вложенные обьекты сохраняются
const person2 = { ...person };

person2.name = "Alice";

console.log(person2.name);
console.log(person.name);
```

Вариант 3

```javascript
const person = {
  name: "Bob",
  age: 25,
};

// Конвертируем в строку, строку в обьект, двойная конвертация
// Ссылки на вложенные обьекты не сохраняются
const person2 = JSON.parse(JSON.stringify(person));

person2.name = "Alice";

console.log(person2.name);
console.log(person.name);
```
