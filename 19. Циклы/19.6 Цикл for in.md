# Цикл for in

```js
// Зарезервированное слово "for", зарезервированное слово "in", блок инструкций (Выполняется для каждого свойства обькта)
for (key in Object) {
  // Действия с каждым свойством обьекта
  // Значения свойства - Object[key]
}
```

```js
for (const key in myObject) {
  console.log(key, myObject[key]);
}
```
