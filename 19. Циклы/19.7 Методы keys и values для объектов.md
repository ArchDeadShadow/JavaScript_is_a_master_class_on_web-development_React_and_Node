# Методы keys и values для объектов

```js
const myObject = {
  x: 10,
  y: true,
  z: "abc",
};

// Получение всех ключей обьекта в виде массива
Object.keys(myObject).forEach((key) => {
  console.log(key, myObject[key]);
});
```

```js
const myObject = {
  x: 10,
  y: true,
  z: "abc",
};

// Получение всех значений свойств обьекта в виде массива
Object.values(myObject).forEach((value) => {
  console.log(value);
});
```
