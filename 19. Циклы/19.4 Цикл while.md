# Цикл while

```js
// Зарезервированное слово "while", цикл выполняется пока условие правдиво, блок инструкций (Может быть ни разу не выполнен)
while (Условие) {
  // Блок кода, выполняемый на каждой итерации
}
```

Пример:

```js
let i = 0;

while (i < 5) {
  console.log(i);
  i++; // Изменение переменной, влияющей на условие
}
```

Бесконечный цикл:

```js
let i = 0;

while (i < 5) {
  console.log(i);
}
```
