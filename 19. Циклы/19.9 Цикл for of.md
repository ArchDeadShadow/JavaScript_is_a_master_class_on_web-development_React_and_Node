# Цикл for of (Появился в ES6)

```js
// Зарезервированное слово "for", зарезервированное слово "of", блок инструкций
for (Element of Iterable) {
  // Действие с определенным элементом
}
```

Пример 1:

```js
const myString = "Hey";

for (const letter of myString) {
  console.log(letter);
}
```

Пример 2:

```js
const myArray = [true, 10, "abc", null];

for (const element of myArray) {
  console.log(element);
}
```

Использовать при итерации всегда метод "forEach"

```js
const myArray = [true, 10, "abc", null];

myArray.forEach((element) => {
  console.log(element);
});
```

"for...of..." - Не для обьектов
