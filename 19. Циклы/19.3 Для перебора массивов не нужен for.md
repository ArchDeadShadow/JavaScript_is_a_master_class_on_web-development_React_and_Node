# Для перебора массивов не нужен for

Цикл можно использовать для массивов - Не рекомендуется!
Используйте функции высшего порядка массивов - "forEach", "map" "reduce"

Цикл "for" для массивов:

```js
const myArray = ["first", "second", "third"];

// Здесь можно использовать свойство "length"
for (let i = 0; i < myArray.length; i++) {
  console.log(myArray[i]);
}
```

Метод массивов "forEach":

```js
const myArray = ["first", "second", "third"];

myArray.forEach((elemnet, index) => {
  console.log(elemnet, index); // Индекс каждого элемента массива
});
```
