# Цикл do while

```js
// Зарезервированное слово "do", блок инструкций (Выполняется хотя бы один раз)
do {
  // Блок кода, выполняемый на каждой итерации
} while (Условие); // Зарезервированное слово "while", цикл выполняется пока условие правдиво
```

Пример 1:

```js
let i = 0;

do {
  console.log(i);
  i++;
} while (i < 5);
```

Пример 2:

```js
let i = 10;

do {
  // Выполнится один раз
  console.log(i);
  i++;
} while (i < 5); // Это условие ложно
```
