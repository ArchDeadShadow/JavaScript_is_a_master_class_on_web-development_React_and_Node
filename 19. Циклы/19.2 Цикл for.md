# Цикл for

```js
// Зарезервированное слово "for", три исходящие инструкции
for (Начальная инструкция; Условие; Итерационное действие) {
	// Блок инструкций, блок кода, выполняемый на каждой итерации
}
```

Пример:

```js
// Зарезервированное слово "for", три исходные инструкции, блок инструкций
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```
