# Операторы

Оператор - это встроеная функция

Виды операторов:

- Операторы
- Унарные и бинарные
- Операнды

Форма записи:

- Инфиксная
- Префиксная и постфиксная

## Операторы символьные

**Арифметические операторы: `+ - * \`**

**Операторы сравнения: `=== !== <= >=`**

**Логические операторы: `! && ||`**

**Оператор присваивания: `=`**

## Операторы текстовые

**Текстовые операторы: `typeof` - Проверить тип значения, `instatof` - Проверить принадлежность обьекта к класу, `new`, `delete` - Удалить свойство в обьекте**

```js
let a, b; // , - Оператор запятая

// = - Оператор присваивание
a = 10;
b = a;

let c = a + b;
console.log(c);
```
