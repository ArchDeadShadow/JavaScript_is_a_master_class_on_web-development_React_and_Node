# Оператор разделения обьекта на свойства

## Оператор ...

```js
const button = {
  width: 200,
  text: "buy",
};

const redButton = {
  ...button,
  // Если у обьекта "button" есть свойство "color", его значение будет перезаписано
  color: "red",
};

console.table(redButton);
```

```js
const buttonInfo = {
  text: "Buy",
};

const buttonStyle = {
  color: "yellow",
  width: 200,
  height: 300,
};

const button = {
  ...buttonInfo,
  ...buttonStyle,
};

console.table(button);
```
