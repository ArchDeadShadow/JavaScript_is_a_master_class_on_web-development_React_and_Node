# Основы React и взаимодействие с DOM

**_Основы React:_**

Document Object Model **_(DOM)_**. Обновление всего DOM - это затратная операция.

Изменение состояния:

- Реальный DOM
- Виртуальный DOM

Две версии виртуального DOM сравниваются с _текущем_ состоянием и _предыдущим_ состоянием. Затем идет определения отличий. Внесение изменений в _реальный_ DOM.

**_Реальный_** и **_Виртуальный_** DOM синхронизированы!
