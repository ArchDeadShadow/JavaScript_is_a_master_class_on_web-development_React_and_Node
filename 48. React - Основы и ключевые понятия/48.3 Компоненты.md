# Компоненты

## Характеристики компонентов

- Компоненты можно использовать **_многократно_**
- Компоненты формируют четкую **_иерархическую_** структуру
- Видом компонентов можна управлять извне с помощью **_свойств_**
- Компоненты могут иметь свое собственное **_состояние_**
- Каждый компонент отвечает за отдельную логическую часть интерфейса
