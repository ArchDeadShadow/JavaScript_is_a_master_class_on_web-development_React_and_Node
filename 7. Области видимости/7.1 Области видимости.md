# Области видимости

Область видимости определяет границы действия переменной.

- Глобальная область видимости, переменные a, b, c
  - Локальная область видимости
    - Локальная область видимости 1, переменные a,c
    - Локальная область видимости 2, переменная b

Области видимости

```javascript
// Глобальная область видимости
// Глобальные переменные
let a;
let b;

function myFn() {
  // Область видимости функции
  // Локальная переменная
  let b;
  a = true;
  b = 10;
  console.log(b); // 10
}

myFn();

console.log(a); // true
console.log(b); // undefined
```
