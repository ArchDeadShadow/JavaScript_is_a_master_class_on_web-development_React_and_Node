# Жизненный цикл переменных

Переменная "b"

```javascript
let a;
// 1. Обьявления "b" в глобальной области видимости. Ее значение undefined
let b;

func myFn () {
	// 3. Обьявление "b" в зоне видимости функции
	let b;
	a = true;
	// 4. Обьявлена ли "b" в рамках функции? Да, присвоение этой переменной "b" значение 10
	b = 10;
	console.log(b); // 10
	// 5. "b" имеет значение 10 в области видимости функции
}

// 2. Вызов myFn
myFn();

console.log(a); // true
console.log(b); // undefined
// 6. "b" все так же имеет значение undefined в глобальной области видимости
```

Перемменая "a"

```javascript
// 1. Обьявления "a" в глобальной области видимости. Ее значение undefined
let a;
let b;

func myFn () {
	let b;
	/*
	* 3. Обьявление "a" в зоне видимости функции? Нет
	* Обьявлена ли "a" во внешней области видимости? ДА
	* Присваивание значения true глобальной переменной "a"
	*/
	a = true;
	b = 10;
	console.log(a); // true
}

// 2. Вызов myFn
myFn();

// 4. "a" имеет значение true
console.log(a); // true
console.log(b); // undefined
```
