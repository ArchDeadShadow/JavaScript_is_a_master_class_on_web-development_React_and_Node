# Типы областей видимости

- Глобальная область видимости
- Область видимости функции
- Область видимости блока: Переменные, обьявленные с помощью let или const внутри блока имеет область видимости, ограниченную эти блоком

Необьявленные переменные. НЕ РЕКОМЕНДУЕТСЯ!

```javascript
function myFn() {
  // Переменная "а" будет автоматически обьявлена в глобальной области видимости
  a = true;
  console.log(a); // true
}

myFn();

console.log(a); // true
```

Правила работы с переменными:

1. Все переменные нужно обьявлять перед их использованием
2. Стараться использовать const везде, где это возможно
3. Внутри функций не изменять переменные с внешних областей видимости
