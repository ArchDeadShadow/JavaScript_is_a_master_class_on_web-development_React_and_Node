# Оператор НЕ

Оператор ! чаще всего используется в условных инструкциях

Примеры с оператором !:

```js
!10;

!0;

!"abc";

!"";

!true;

!undefined;
```

Отрицание отрицания !!:

```js
!!10;

!!0;

!!"abc";

!!"";

!!true;

!!undefined;
```

Позволяет проверить ложность значения

```js
let isObj = {};

!!isObj;
```
