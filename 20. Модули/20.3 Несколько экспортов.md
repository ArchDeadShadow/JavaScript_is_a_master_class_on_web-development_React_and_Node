# Несколько экспортов

Node.js - поддерживает ES6 модули, начиная с версии 13

```js
// moduleOne.js
const one = 1;
const two = "two";

export { one, two };
```

```js
// moduleTwo.js
// Имена переменных должны совпадать
import { one as oneRenamed, two } from "./moduleOne.js"; // Переменование импортов, при импорте можно переименовывать переменные

console.log(oneRenamed);
console.log(two);
```
