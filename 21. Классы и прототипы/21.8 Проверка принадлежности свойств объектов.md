# Проверка принадлежности свойств объектов

Проверка принадлежности свойств экземпляру обьекта:

```js
class Comment {
  constructor(text) {
    this.text = text;
    this.votesQty = 0;
  }

  upvote() {
    this.votesQty += 1;
  }
}

const firstComment = new Comment("First comment");

firstComment.hasOwnProperty("text");
firstComment.hasOwnProperty("votesQty");
firstComment.hasOwnProperty("upvote");
firstComment.hasOwnProperty("hasOwnProperty");
```
