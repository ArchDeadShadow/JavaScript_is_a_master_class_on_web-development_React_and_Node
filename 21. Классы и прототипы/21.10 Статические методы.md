# Статические методы

```js
class Comment {
  constructor(text) {
    this.text = text;
    this.votesQty = 0;
  }

  upvote() {
    this.votesQty += 1;
  }

  static mergeComments(first, second) {
    return `${first} ${second}`;
  }
}

// Метод доступен как свойство класса и не наследуется экземплярами класса
Comment.mergeComments("First comment", "Second comment");
```
