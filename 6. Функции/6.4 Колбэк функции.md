# Колбэк функции

```javascript
function anotherFunction() {
  // Дейсивия...
}

function fnWithCallback(callbackFunction) {
  // В теле этой функции вызывается колбэк функция
  callbackFunction();
}

fnWithCallback(anotherFunction);
```

Колбэк функция вызывается внутри другой функции.

```javascript
function printMyName() {
  console.log("Sergey");
}

/*
 * Вызов колбэк функции через встроенную JS функцию.
 * Функция "printMyName" будет вызвана через 1000 миллисекунд
 */
setTimeout(printMyName, 1000);
```
