# Передача значения по ссылке

```javascript
const personOne = {
  name: "Bob",
  age: 21,
};

// Функция мутирует внешний обьект
function increasePersonOne(person) {
  person += 1;
  return person;
}

// Передача обьекта по ссылке
increasePersonOne(personOne);
console.log(personOne.age);
```

Внутри функции не рекомендуется мутировать внешние обьекты.

```javascript
// Создание копии обьекта:
const personOne = {
  name: "Bob",
  age: 21,
};

function increasePersonOne(person) {
  const updatePerson = Object.assign({}, person);
  updatePerson.age += 1;
  return updatePerson;
}

const updatePersonOne = increasePersonOne(personOne);
console.log(personOne.age); // 21
console.log(updatePersonOne.age); // 22
```
