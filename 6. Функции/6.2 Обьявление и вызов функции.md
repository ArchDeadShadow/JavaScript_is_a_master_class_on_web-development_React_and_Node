# Обьявление и вызов функции

```javascript
// Обьявление функции.
// Имя функции myFn.
// Параметры a, b - это как переменные, которые действуют только внутри функции.
function myFn(a, b) {
  // Тело функции.
  let c;
  a = a + 1;
  c = a + b;
  // Результат возвращения и завершения функции:
  return c;
}

// Вызов функции. Аргументы 10, 3
myFn(10, 3);

const result = myFn(10, 3);
console.log(result);
```

Функция возвращает undefined, если нет инструкции return.

Параметры, return, аргументы - опциональны.

```javascript
// Самая короткая функция
// Обьявление функции
function myFn() {}

// Вызов функции
myFn(); // undefined
```
