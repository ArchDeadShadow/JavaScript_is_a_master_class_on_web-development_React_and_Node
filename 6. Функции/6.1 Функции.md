# Функции

Функция - это блок кода, который можно выполнять многократно

```javascript
let a = 5;
let b = 3;

// Функция с параметрами a и b
function sum(a, b) {
  // Тело функции
  const c = a + b;
  console.log(c);
}

sum(a, b);

a = 8;
b = 12;

sum(a, b);
```

Функция может быть:

- именованной
- присвоена переменной
- анонимной
- аргументом при вызове другой функции
- функция свойства (метода) обьекта

Функция - это обьект

```javascript
function myFn(a, b) {
  let c;
  a = a + 1;
  c = a + b;
  return c;
}

console.dir(myFn);
```
