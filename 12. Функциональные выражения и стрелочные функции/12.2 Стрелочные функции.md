# Стрелочные функции

```js
// В стрелочной функции нет имени, сразу указываются параметры
(a, b) => {
  // Тело функции в фигурных скобках
  let c;

  a = a + 1;
  c = a + b;

  return c; // Результат
};
```

**_Стрелочная функция - это выражение_**

**_Стрелочные функции всегда анонимны_**

Как дать имя стрелочной функции? Путем присваивания переменной

```js
// Обьявляется переменная. Присваивание переменной анонимное функциональное выражение
const myFunction = (a, b) => {
  let c;

  a = a + 1;
  c = a + b;

  return c;
};

myFunction(5, 3);
```

Стрелочная функция в вызове другой функции

```js
setTimeout(() => {
  console.log("Отложенное сообщение");
}, 1000);
```

Сокращение в стрелочных функциях

Вариант 1:

```js
// Если один параметр, то круглые скобки можно опустить, но лучше всего добавить скобки
(a) => {
  // Тело функции
};
```

Вариант 2:

```js
// Фигурные скобки можно опустить если тело функции состоит из одного выражения
(a, b) => a + b;
// В этом случае стрелочная функция неявно возвращает результат выражения (Неявный возврат значения)
```
