# Значения параметров функции по умолчанию

Пример 1:

```js
// Присваивания параметру multiplayer значения по умолчанию 1
function multByFactor(value, multiplayer = 1) {
  return value * multiplayer;
}

multByFactor(10, 2);
multByFactor(5);
```

Пример 2:

```js
// Значения по умолчанию вычисляются в момент вызова функции
const newPost = (post, addedAt = Date()) => ({
  // Не явный возврат обьекта
  ...post,
  addedAt,
});

const firstPerson = {
  id: 1,
  author: "Sergey",
};

newPost(firstPerson);
```

Выполненное задание урока:

```js
const newPost = (post, addedAt = Date()) => {
  return {
    ...post,
    addedAt,
  };
};

const firstPerson = {
  id: 1,
  author: "Sergey",
};

console.log(newPost(firstPerson));
```
